Drop Nox — Cloud Storage Backend (Google Drive–like System)

Drop Nox is a cloud storage application inspired by Google Cloud / Google Drive, focused on backend system design and file-management logic. The project allows users to manage files and folders through a structured API, demonstrating real-world backend engineering concepts such as hierarchical data modeling, soft deletion, and recovery workflows.

Core Features

Users can:

Upload files

Upload folders

Create folders

Move files and folders between directories

Soft delete files and folders (non-destructive deletion)

Recover deleted files and folders
